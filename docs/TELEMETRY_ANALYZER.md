# 📊 ASCEND Telemetry Analyzer

A comprehensive web-based dashboard for visualizing and analyzing drone flight telemetry data from ASCEND missions.

## 🚀 Features

- **Real-time Data Loading**: Automatically detects and loads recent log files
- **Multi-axis Visualization**: Interactive Plotly charts for all telemetry parameters
- **Section Coverage**:
  - 📍 **Position & Flight Path**: Altitude profile, climb rate, 2D trajectory
  - 🎛️ **Vehicle Dynamics**: Attitude angles (roll, pitch, yaw), vibration analysis
  - 🔋 **System Health**: Battery voltage, current draw, CPU load
  - 🎮 **Control Inputs**: RC channel history
  - 📈 **Statistics**: Descriptive statistics for all numeric parameters
  - 🔍 **Raw Data Explorer**: Search and inspect raw CSV data

## 📋 Prerequisites

Install dependencies from `requirements.txt`:

```bash
pip install -r requirements.txt
```

Or install Streamlit and Plotly manually:

```bash
pip install streamlit plotly pandas numpy
```

## 🏃 Running the Dashboard

```bash
streamlit run telemetry_analyzer.py
```

The dashboard will open in your browser at `http://localhost:8501`

### Alternative (if running remotely):
```bash
streamlit run telemetry_analyzer.py --server.port 8501 --server.address 0.0.0.0
```

Then access at `http://<your-ip>:8501`

## 📂 Data Format

The analyzer supports CSV files with the following telemetry columns:

### Expected Column Names (case-flexible):
- **Position**: `Timestamp`, `Baro_Alt_m`, `Lidar_Dist_m`, `Climb_Rate_ms`
- **Optical Flow**: `Opt_Flow_X`, `Opt_Flow_Y`, `Opt_Flow_Qual`
- **Attitude**: `Roll_deg`, `Pitch_deg`, `Yaw_deg`
- **Battery**: `Batt_Voltage`, `Batt_Current_A`, `Batt_Remain_Pct`
- **Dynamics**: `Vibration_X`, `Vibration_Y`, `Vibration_Z`
- **System**: `CPU_Load_Pct`, `Flight_Mode`, `Armed`
- **Control**: `RC1_Roll`, `RC2_Pitch`, `RC3_Throttle`, `RC4_Yaw`

### Example CSV Structure:
```
Timestamp,Flight_Mode,Roll_deg,Pitch_deg,Yaw_deg,Opt_Flow_X,Opt_Flow_Y,Opt_Flow_Qual,Lidar_Dist_m,Baro_Alt_m,Climb_Rate_ms,Batt_Voltage,Batt_Current_A,Batt_Remain_Pct,RC1_Roll,RC2_Pitch,RC3_Throttle,RC4_Yaw,Vibration_X,Vibration_Y,Vibration_Z,CPU_Load_Pct
1234567890.1,ALT_HOLD,0.1,0.2,45.5,0.05,-0.02,200,1.23,0.50,0.10,12.45,5.20,85,1500,1500,1680,1500,0.001,0.002,0.001,45.0
```

## 📊 Dashboard Sections

### 1. **Flight Summary**
- Flight duration
- Maximum altitude reached
- Minimum battery voltage
- Total data points collected

### 2. **Position & Flight Path**
- Altitude profile with envelope
- Vertical velocity (climb rate)
- 2D trajectory visualization (Optical Flow + LiDAR)

### 3. **Vehicle Dynamics**
- Attitude changes over time (roll, pitch, yaw)
- Vibration signature analysis (X, Y, Z axes)

### 4. **System Health**
- Battery voltage and remaining capacity
- Power consumption (current draw)
- CPU/system load

### 5. **Control Inputs**
- RC channel values (PWM 1000-2000)
- Control history visualization

### 6. **Raw Data Explorer**
- Search specific telemetry columns
- Export subsets of data
- Statistical breakdown

## 🛠️ Customization

### Add Custom Metrics
Edit line ~80 to add your own column mappings:
```python
alt_cols = [col for col in df.columns if 'alt' in col.lower()]
```

### Change Color Schemes
Modify Plotly color specifications (e.g., line 159):
```python
line=dict(color='green', width=2)
```

### Add Export Functionality
Add to end of script:
```python
if st.button("📥 Export to CSV"):
    csv = df.to_csv(index=False)
    st.download_button("Download CSV", csv, "telemetry_export.csv")
```

## 📝 Log File Generation

This analyzer works with CSV logs generated by:
- `ascend_blackbox.py` - Comprehensive flight log
- `log_flight_data.py` - Optical flow data
- `pc_data_visualizer.py` - Real-time telemetry capture
- `ascend_tui.py` - Mission control session logs

Example: Generate a log file first:
```bash
python src/ascend_blackbox.py  # Logs to full_flight_log_<timestamp>.csv
```

Then analyze it:
```bash
streamlit run telemetry_analyzer.py
```

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| "No log files found" | Manually upload a CSV or generate one with `ascend_blackbox.py` |
| Chart not displaying | Check that required columns exist in CSV (case-insensitive) |
| Import errors | Run `pip install -r requirements.txt` |
| Remote access not working | Use `--server.address 0.0.0.0` flag |
